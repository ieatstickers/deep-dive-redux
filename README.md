# Deep Dive: Redux


## How to Build

#### 1. Install Dependencies

```
npm install
```

#### 2. Run Webpack

Note: The `WATCH` flag is optional 
```
WATCH=true webpack --config webpack.config.js
```

#### 3. Serve the Public Directory

```
cd ./public
php -S localhost:2000
```

#### 4. View Site

Go to `localhost:2000` in any browser
